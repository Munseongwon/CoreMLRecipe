# CoreMLRecipe
CoreML을 활용하여 레시피를 만들어보았습니다

# 어떤 언어를 사용하였는가?
 - Swift
 
# 화면 구조는 어떻게 되는가?
 - Splash 화면
 - ImagePicker를 통한 이미지 선택 및 이미지 모델링 작업(분류: 양파 혹은 토마토)
 - 이미지가 선택되면, recipe 버튼을 클릭하면 각 모델마다 레시피가 분류되어 나오게 됨
